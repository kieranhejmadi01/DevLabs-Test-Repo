# Arm Developer Labs

This repository contains the source files for the Arm Developer Labs static website, focused on challenge-based technical content for Arm software developers.

Content in this repository is centered on Arm Developer Challenges and related challenge resources.

## Contributing

Contributions are welcome for challenge content, challenge improvements, and repository fixes relevant to Arm Developer Labs.

- Open an issue in this repository for bugs or suggestions.
- Submit a pull request for challenge updates or new challenge material.

All site content, including new contributions, is licensed under a [Creative Commons Attribution 4.0 International license](https://creativecommons.org/licenses/by/4.0/).

## AI Development Tools

When using AI coding assistants (GitHub Copilot, Amazon Q, Gemini, Cursor, etc.), refer to `.github/copilot-instructions.md` for project-specific guidelines.

## Directory Structure

This site is built with [Hugo](https://gohugo.io/) and uses the Arm design Hugo theme (`arm-design-system-hugo-theme`).

- /content
  - challenge content and challenge pages
- /themes
  - templates and rendering logic for site pages
- /tools
  - scripts for maintenance and checks
- LICENSE files
  - licensing information
- config.toml
  - top-level site configuration
